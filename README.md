# MediMemory AI

MediMemory AI is a demonstration medical record assistant designed to showcase the integration of FastAPI, SQLAlchemy, and document processing for healthcare data management.

## ⚠️ Safety Warning
**Demo application using fictional medical records only. Not intended for clinical use.** 
This software is provided for demonstration purposes and does not comply with HIPAA or other regional medical data privacy regulations out-of-the-box. Never upload real Protected Health Information (PHI) to this demo.
